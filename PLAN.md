# Add run deletion & improve rest day card

**Features**

- **Delete runs from history** — Long-press or swipe on any run in the history list to reveal a delete option, with a confirmation dialog to prevent accidental deletions
- **Enhanced rest day card** — Rest days get a more visually interesting card with a calming gradient background, recovery tips, and a subtle animation

**Design**

*Run Deletion*

- Long-press on a run card shows a confirmation dialog ("Delete this run?") with Cancel and Delete buttons
- A small trash icon appears on hover/press to hint at the action
- Deleted runs are removed from history and stats are updated accordingly

*Rest Day Card*

- Dark card with a subtle blue-to-teal gradient border/accent instead of the current plain flat look
- Starry/nighttime-inspired background pattern using small decorative dots
- A motivational recovery quote that rotates each day (e.g. "Muscles grow during rest, not during the workout")
- Gentle pulse animation on the moon icon to add life to the card
- "Next workout" info stays visible with slightly more prominence
